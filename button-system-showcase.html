Integration & Usage

React/TSX Usage

// Import component
import { Button } from '@/components/ui';

// Primary action
<Button variant="primary">Connect Account</Button>

// Secondary action
<Button variant="secondary">Cancel</Button>

// Danger action
<Button variant="danger">Delete</Button>

// Low emphasis
<Button variant="ghost">Skip</Button>

// With size
<Button variant="primary" size="lg">Large</Button>

// Loading state
<Button variant="primary" isLoading={true}>Connecting...</Button>

Vanilla HTML Usage

// Primary
<button class="button button-primary button-md">Connect</button>

// Secondary
<button class="button button-secondary button-md">Cancel</button>

// Danger
<button class="button button-danger button-md">Delete</button>

// Ghost
<button class="button button-ghost button-md">Skip</button>

Accessibility (WCAG AA ✅)

✅ What's Included

  • Minimum 44×44px hit area (mobile safe)
  • ≥4.5:1 color contrast all states
  • Visible focus outline (2px + 2px offset)
  • Disabled state both visual + semantic
  • No hover-only content
  • Keyboard accessible (Tab, Enter/Space)
  • Respects prefers-reduced-motion

📋 Testing Checklist

  • Tab through all buttons → focus visible?
  • Press Enter/Space → button activates?
  • Screen reader → button label read?
  • Color contrast tool → ≥4.5:1?
  • prefers-reduced-motion: reduce → animations off?
  • Mobile → hit area ≥44px?
  • Dark mode → contrast maintained?
