# Platform Multi Publisher Landing - Fallback Mode v2

Assumption: this round is for the public SEO and conversion homepage. A new visitor should understand in five seconds that Platform Multi Publisher publishes videos to YouTube, TikTok and Instagram from one workspace, through the flow: connect accounts -> create campaign -> publish -> track status.

The attached images are treated as mood references only: chunky display typography, dot-matrix counters, maximalist poster rhythm and friendly single-line mascot energy. Their text is not copied.
