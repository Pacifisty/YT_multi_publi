# PMP Public Landing - Fallback Mode

Assumption: this is the public SEO/conversion homepage for Platform Multi Publisher. The user must understand in 5 seconds that the product publishes videos to YouTube, TikTok and Instagram from one workspace through connected accounts, campaigns, media library, publishing queue and reports.

The attached images are used only as visual references. Their text is not copied. The approved visual anchor is `Demo 04 - Psychedelic Signal`: loud creator energy, rounded poster frames, saturated color, animated rays, cloud/signal motifs, and tactile printed texture.
